# Exploratory Data Analysis on Amazon Reviews

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Amazon product reviews to uncover customer behavior, review trends, and product insights.

## 📊 Features

* Data cleaning and preprocessing
* Missing value analysis
* Rating distribution analysis
* Product category analysis
* Review length analysis
* Trend and pattern identification

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📈 Key Insights

* Most products received highly positive ratings.
* Electronics-related categories dominated customer engagement.
* Customer reviews varied significantly in length and detail.
